# City Raft
Imlpementation of Raft in Rust for learning.

## Purpose
A step-by-step implementation  of:
- What Raft is
- How it's implemented
- Async integration

## Installing
`cargo install`

## Running
`cargo run`

## Dependencies
- Tokio
